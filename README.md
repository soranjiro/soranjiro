## ğŸš€ Quick Start

### 1. Data Fetching

```bash
export GITHUB_TOKEN="your_github_token"
node src/fetch-data.mjs
```

Fetches data from:
- GitHub GraphQL API (profile, repos, languages)
- GitHub REST Search API (accurate contribution counts)
- GitHub Models API (AI-generated summaries)

Output: `output/data.json`

### 2. Dashboard Generation

```bash
node src/generate-dashboard.mjs
```

Generates: `output/dashboard.html`

### 3. View Dashboard

```bash
cd output && python3 -m http.server 8765
# Open http://localhost:8765/dashboard.html
```

## ğŸ“Š Data Visualization

### Radar Chart
- **Purpose**: Multi-dimensional activity metrics (Commits, PRs, Reviews, Issues, Comments)
- **Innovation**: Logarithmic scale transformation for handling 100x range differences
- **Styling**: Amber accent color, dark tooltips (#1a1c22)

### Bar Chart
- **Purpose**: Yearly contribution trends with public/private split
- **Data**: Adds estimated private commits based on GitHub stats

### Line Chart
- **Purpose**: Language skill evolution over time
- **Data**: Absolute values normalized to max for indicator bar widths
